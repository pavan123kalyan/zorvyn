# zorvyn
🚀 Project Title: Zorvyn – Personal Finance Dashboard
🛠️ Tech Stack:
React
JavaScript
Vite
CSS
✨ Features:
Add, edit, and manage transactions
View balance trends over the last 30 days
Analyze spending category-wise with donut charts
Interactive financial insights and summary cards
Responsive UI with dark and light theme support
Real-time transaction updates with charts and analytics
📌 Description:
A modern personal finance dashboard that helps users track income, expenses, balance trends, and spending patterns through an interactive and visually appealing interface.
